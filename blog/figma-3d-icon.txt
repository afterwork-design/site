                    <h1 id="Figma 设计轻 3D 效果图标教程">
                        <a href="#Figma 设计轻 3D 效果图标教程">Figma 设计轻 3D 效果图标教程</a>
                    </h1>
                <p><img src="../assests/blog/image/3d-icon-cover.jpg" alt="cover"></p>
<p>源文件：<a href="https://www.figma.com/community/file/1350347700225468380/3d">https://www.figma.com/community/file/1350347700225468380/3d</a></p>

                    <h2 id="一、挤出效果">
                        <a href="#一、挤出效果">一、挤出效果</a>
                    </h2>
                <p>设计图标不同于建模，不需要十分逼真，提取主要特征即可，仅使用颜色、阴影、模糊来实现 3D 挤出效果。</p>
<p>以一个 3 层的图标为例，分别说明如何设置参数。</p>
<p><img src="../assests/blog/image/extrude.jpg" alt="extrude"></p>

                    <h3 id="背景层">
                        <a href="#背景层">背景层</a>
                    </h3>
                <p>背景层最容易设置，可以填充最浅色的纯色，不添加任何装饰</p>
<p><img src="../assests/blog/image/background.jpg" alt="background"></p>

                    <h3 id="中层">
                        <a href="#中层">中层</a>
                    </h3>
                <ol>
<li><p>中层需要填充一个渐变色，推荐使用同类色渐变，实践下来，将渐变两端颜色的 H 值设置为相差 20 以内会比较和谐，也可以参考一些<a href="https://afterwork-design.github.io/color-card/gradient">色彩网站</a>中配好的颜色。</p>
</li>
<li><p>设置浅色内阴影，模拟亮部，所以需要设置为从上往下投影。</p>
</li>
<li><p>设置深色内阴影，模拟暗部，所以需要设置为从下往上投影。</p>
</li>
<li><p>设置深色外阴影，模拟因与背景有一段距离而形成的阴影。</p>
</li>
</ol>
<p><img src="../assests/blog/image/extrude-m.jpg" alt="extrude-m">
<img src="../assests/blog/image/extrude-m-effects.jpg" alt="extrude-m-effects"></p>

                    <h3 id="顶层">
                        <a href="#顶层">顶层</a>
                    </h3>
                <p>与中层同理，但可以减少一些细节，仅需添加白色填充和深色外阴影。</p>
<p><img src="../assests/blog/image/extrude-top.jpg" alt="extrude-top"></p>

                    <h2 id="二、膨胀效果">
                        <a href="#二、膨胀效果">二、膨胀效果</a>
                    </h2>
                <p>同样是使用颜色、阴影、模糊参数来实现。</p>
<p><img src="../assests/blog/image/inflate.jpg" alt="inflate"></p>

                    <h3 id="背景层">
                        <a href="#背景层">背景层</a>
                    </h3>
                <p>膨胀效果与挤出效果的背景层设置方法相同。</p>
<p><img src="../assests/blog/image/background.jpg" alt="background"></p>

                    <h3 id="中层">
                        <a href="#中层">中层</a>
                    </h3>
                <ol>
<li><p>渐变色填充</p>
</li>
<li><p>浅色内阴影</p>
</li>
<li><p>深色内阴影，范围稍微设置大一些</p>
</li>
<li><p>深色外阴影</p>
</li>
</ol>
<p><img src="../assests/blog/image/inflate-m.jpg" alt="inflate-m">
<img src="../assests/blog/image/inflate-m-effects.jpg" alt="inflate-m-effects"></p>

                    <h3 id="顶层">
                        <a href="#顶层">顶层</a>
                    </h3>
                <ol>
<li><p>白色填充</p>
</li>
<li><p>深色内阴影</p>
</li>
<li><p>深色外阴影</p>
</li>
</ol>
<blockquote>
<p>阴影的取色参考中层，色相不变，明度降低</p>
</blockquote>
<p><img src="../assests/blog/image/inflate-top.jpg" alt="inflate-top"></p>

                    <h2 id="三、磨砂玻璃效果">
                        <a href="#三、磨砂玻璃效果">三、磨砂玻璃效果</a>

                <p>磨砂玻璃效果以 4 层的图标为例。</p>
<p><img src="../assests/blog/image/ground-glass.jpg" alt="ground-glass"></p>

                    <h3 id="背景">
                        <a href="#背景">背景</a>
                    </h3>
                <p>同上，可深可浅。</p>
<p><img src="../assests/blog/image/background.jpg" alt="background"></p>

                        <a href="#中 2 层">中 2 层</a>
