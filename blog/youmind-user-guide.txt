                    <h1 id="youmind-界面使用指南-对话资料作品三大模块详解">
                        <a href="#youmind-界面使用指南-对话资料作品三大模块详解">YouMind 界面使用指南 - 对话、资料、作品三大模块详解</a>
                    </h1>
                <p><img src="https://youmind.com/_next/image?url=https%3A%2F%2Fassets.youmind.com%2Fcraft-page-covers%2Fcover-19.png&w=3840&q=75" alt="youmind"></p>
<p><a href="https://youmind.com/zh-CN/overview">YouMind</a> 是一个 AI 驱动的个人知识工作台。如果你是初次使用它，迷失在了 YouMind 令人眼花缭乱的卡片界面中，这篇文章或许对你有用。</p>

                    <h2 id="youmind-界面的核心理解">
                        <a href="#youmind-界面的核心理解">YouMind 界面的核心理解</a>
                    </h2>
                <p>首先要理解一个关键点：<strong>YouMind 的界面不是传统的&quot;三个独立标签页&quot;，而是一个集成的工作空间</strong>。</p>
<p>当你打开一个 Board（看板）时，你看到的是<strong>三栏布局</strong>：</p>
<ul>
<li><p>左侧：<strong>Materials（资料）</strong></p>
</li>
<li><p>中间：<strong>Crafts（作品）</strong></p>
</li>
<li><p>右侧：<strong>Tools/Chat（工具/对话）</strong></p>
</li>
</ul>
<p>这三栏是<strong>同时存在、互相关联</strong>的，不需要来回切换标签页。这就是 YouMind 被称为&quot;集成创作环境&quot;（Integrated Creation Environment）的原因。</p>

                    <h2 id="左侧materials（资料区）">
                        <a href="#左侧materials（资料区）">左侧：Materials（资料区）</a>
                    </h2>
                
                    <h3 id="这里是什么">
                        <a href="#这里是什么">这里是什么？</a>
                    </h3>
                <p>Materials 是你的&quot;素材库&quot;，所有收集的内容都会显示在这里：</p>
<p><strong>支持的内容类型：</strong></p>
<ul>
<li><p>网页片段（通过浏览器插件保存的 Snips）</p>
</li>
<li><p>YouTube 视频（自动生成完整文字稿）</p>
</li>
<li><p>播客音频（支持 m4a 格式）</p>
</li>
<li><p>PDF 文档</p>
</li>
<li><p>图片</p>
</li>
<li><p>文章链接</p>
</li>
<li><p>你手动添加的笔记（Notes）</p>
</li>
</ul>
<p><strong>实际显示效果：</strong></p>
<ul>
<li><p>每个资料显示为一个卡片</p>
</li>
<li><p>自动生成 AI 摘要</p>
</li>
<li><p>视频/播客会显示时间轴，可以跳转到具体段落</p>
</li>
<li><p>PDF 可以高亮标注重点</p>
</li>
</ul>

                    <h3 id="你可以在这里做什么">
                        <a href="#你可以在这里做什么">你可以在这里做什么？</a>
                    </h3>
                <p><strong>1. 阅读与研究</strong></p>
<ul>
<li><p>点击任何资料，进入&quot;干净无干扰&quot;的阅读模式</p>
</li>
<li><p>视频/播客会自动转为文字稿，可以搜索关键词</p>
</li>
<li><p>高亮重要段落（Picks），这些高亮会单独保存</p>
</li>
</ul>
<p><strong>2. 添加笔记</strong></p>
<ul>
<li><p>对任何资料添加你的想法（Notes）</p>
</li>
<li><p>标记哪些内容对你的项目有价值</p>
</li>
</ul>
<p><strong>3. 生成音频摘要</strong></p>
<ul>
<li><p>使用&quot;Listen&quot;功能，AI 会生成 3 分钟音频概述</p>
</li>
<li><p>可以在通勤时听，加深理解</p>
</li>
</ul>
<p><strong>4. 与 AI 对话</strong></p>
<ul>
<li><p>直接在资料上提问</p>
</li>
<li><p>比如：&quot;这篇文章的核心论点是什么？&quot;</p>
</li>
<li><p>&quot;这个视频里提到的案例有哪些？&quot;</p>
</li>
</ul>

                    <h2 id="中间crafts（作品区）">
                        <a href="#中间crafts（作品区）">中间：Crafts（作品区）</a>
                    </h2>
                
                    <h3 id="这里是什么-2">
                        <a href="#这里是什么-2">这里是什么？</a>
                    </h3>
                <p>Crafts 是你的&quot;创作空间&quot;，所有输出内容都在这里完成。这不是一个简单的编辑器，而是一个<strong>支持 AI 协作的写作环境</strong>。</p>
<p><strong>你可以创建的内容类型：</strong></p>
<ul>
<li><p>文章/报告</p>
</li>
<li><p>演讲稿</p>
</li>
<li><p>播客脚本</p>
</li>
<li><p>视频脚本</p>
</li>
<li><p>研究摘要（Briefing digest）</p>
</li>
<li><p>任何结构化文档</p>
</li>
</ul>

                    <h3 id="核心功能">
                        <a href="#核心功能">核心功能</a>
                    </h3>
                <p><strong>1. 从模板开始</strong></p>
<ul>
<li><p>点击&quot;New Craft&quot;时，可以选择模板</p>
</li>
<li><p>比如&quot;Briefing digest&quot;：自动总结 Board 内所有资料</p>
</li>
<li><p>一键导入，生成结构化内容</p>
</li>
</ul>
<p><strong>2. 拖拽式创作</strong></p>
<ul>
<li><p>直接从左侧 Materials <strong>拖拽关键点到 Crafts 中间</strong></p>
</li>
<li><p>YouMind 会自动提供写作结构</p>
</li>
<li><p>你可以基于这个结构展开写作</p>
</li>
</ul>
<p><strong>3. 富文本编辑</strong></p>
<ul>
<li><p>支持 Markdown 格式</p>
</li>
<li><p>可以插入图片、链接、列表</p>
</li>
<li><p>实时预览效果</p>
</li>
</ul>
<p><strong>4. AI 协作写作</strong></p>
<ul>
<li><p>任何一段内容都可以要求 AI 重写、扩展或改写</p>
</li>
<li><p>可以分段生成，而不是一次性生成整篇</p>
</li>
<li><p>生成的内容<strong>完全可编辑</strong>，不是只读的</p>
</li>
</ul>
<p><strong>5. 引用管理</strong></p>
<ul>
<li><p>写作时可以添加来自 Materials 的引用</p>
</li>
<li><p>自动关联到原始资料</p>
</li>
<li><p>保持内容的可追溯性</p>
</li>
</ul>

                    <h2 id="右侧toolschat（工具对话区）">
                        <a href="#右侧toolschat（工具对话区）">右侧：Tools/Chat（工具/对话区）</a>
                    </h2>
                
                    <h3 id="这里是什么-3">
                        <a href="#这里是什么-3">这里是什么？</a>
                    </h3>
                <p>这是你的&quot;AI 助手面板&quot;，提供多种工具和对话功能。</p>

                    <h3 id="核心功能-2">
                        <a href="#核心功能-2">核心功能</a>
                    </h3>
                <p><strong>1. AI Chat（智能对话）</strong></p>
<p>你可以选择不同的 AI 模型：</p>
<ul>
<li><p>YouMind AI（默认）</p>
</li>
<li><p>Claude</p>
</li>
<li><p>ChatGPT (GPT-4)</p>
</li>
<li><p>可以在同一个 Board 中切换模型使用</p>
</li>
</ul>
<p><strong>对话的智能性：</strong></p>
<ul>
<li><p><strong>自动关联 Board 上下文</strong>：AI 知道你正在研究什么主题，看到了哪些资料</p>
</li>
<li><p><strong>多资料问答</strong>：可以同时引用多个 Materials 提问</p>
<ul>
<li><p>例如：&quot;对比这三篇文章的观点差异&quot;</p>
</li>
<li><p>&quot;基于我保存的这些资料，总结主要趋势&quot;</p>
</li>
</ul>
</li>
<li><p><strong>实时更新</strong>：当你添加新资料时，AI 的回答会包含最新内容</p>
</li>
</ul>
<p><strong>2. Ask（快速提问）</strong></p>
<ul>
<li><p>点击右侧的&quot;Ask&quot;按钮</p>
</li>
<li><p>输入问题，AI 会基于当前 Board 的所有资料回答</p>
</li>
<li><p>可以选择使用哪个 AI 模型</p>
</li>
</ul>
<p><strong>3. 编辑工具</strong><br>在 Crafts 写作时，右侧会显示：</p>
<ul>
<li><p><strong>Rewrite（重写）</strong>：改写当前段落</p>
</li>
<li><p><strong>Expand（扩展）</strong>：增加更多细节</p>
</li>
<li><p><strong>Summarize（总结）</strong>：压缩内容</p>
</li>
<li><p><strong>Add citations（添加引用）</strong>：自动关联 Materials</p>
</li>
</ul>

                    <h2 id="三大模块如何协同工作">
                        <a href="#三大模块如何协同工作">三大模块如何协同工作</a>
                    </h2>
                
                    <h3 id="典型的工作流程">
                        <a href="#典型的工作流程">典型的工作流程</a>
                    </h3>
                <p><strong>场景：写一篇关于&quot;远程工作趋势&quot;的文章</strong></p>
<p><strong>第 1 步：收集资料（Materials）</strong></p>
<ol>
<li><p>使用浏览器插件保存 5 篇相关文章</p>
</li>
<li><p>保存 2 个 YouTube 视频（自动转文字稿）</p>
</li>
<li><p>上传一份行业报告 PDF<br>→ 这些都出现在左侧 Materials 区</p>
</li>
</ol>
<p><strong>第 2 步：研究与提问（Chat/Tools）</strong></p>
<ol>
<li><p>在右侧 Chat 输入：&quot;总结这些资料中提到的远程工作的主要优势和挑战&quot;</p>
</li>
<li><p>AI 自动分析所有 Materials，给出结构化答案</p>
</li>
<li><p>继续追问：&quot;有哪些数据支持这些观点？&quot;<br>→ AI 回答会引用具体的资料来源</p>
</li>
</ol>
<p><strong>第 3 步：创作输出（Crafts）</strong></p>
<ol>
<li><p>在中间 Crafts 区点击&quot;New Craft&quot;</p>
</li>
<li><p>选择&quot;Article&quot;模板（或从空白开始）</p>
</li>
<li><p><strong>从左侧拖拽</strong>重要的 Picks（高亮段落）到 Crafts</p>
</li>
<li><p>AI 根据这些素材生成文章大纲</p>
</li>
<li><p>你逐段编辑和扩展内容</p>
</li>
<li><p>使用右侧工具重写某些段落</p>
</li>
<li><p>添加引用链接到原始 Materials<br>→ 完成一篇有理有据的文章</p>
</li>
</ol>
<p><strong>第 4 步：迭代优化</strong></p>
<ol>
<li><p>在 Crafts 中选中某段内容</p>
</li>
<li><p>点击右侧&quot;Rewrite&quot;，让 AI 提供改写版本</p>
</li>
<li><p>或者在 Chat 问：&quot;这一段怎么写更有说服力？&quot;</p>
</li>
<li><p>AI 给出建议，你决定是否采纳</p>
</li>
</ol>

                    <h2 id="界面的关键设计理念">
                        <a href="#界面的关键设计理念">界面的关键设计理念</a>
                    </h2>
                
                    <h3 id="1-三栏同时可见的好处">
                        <a href="#1-三栏同时可见的好处">1. 三栏同时可见的好处</a>
                    </h3>
                <p><strong>传统工具的问题：</strong></p>
<ul>
<li><p>收集资料用一个工具（如 Pocket）</p>
</li>
<li><p>写作用另一个工具（如 Google Docs）</p>
</li>
<li><p>AI 对话用第三个工具（如 ChatGPT）</p>
</li>
<li><p>来回切换，思路容易断</p>
</li>
</ul>
<p><strong>YouMind 的解决方案：</strong></p>
<ul>
<li><p>资料、创作、AI 助手<strong>同屏显示</strong></p>
</li>
<li><p>左边看资料，中间写作，右边问 AI</p>
</li>
<li><p>不需要切换窗口，工作流完全连贯</p>
</li>
</ul>

                    <h3 id="2-信息流动的顺畅性">
                        <a href="#2-信息流动的顺畅性">2. 信息流动的顺畅性</a>
                    </h3>
                <p><strong>从左到中：素材 → 作品</strong></p>
<ul>
<li><p>拖拽资料到 Crafts</p>
</li>
<li><p>点击&quot;Import&quot;一键导入所有重点</p>
</li>
<li><p>资料自动成为写作的支撑</p>
</li>
</ul>
<p><strong>从左到右：素材 → AI 分析</strong></p>
<ul>
<li><p>Chat 自动读取 Materials</p>
</li>
<li><p>提问时 AI 会引用具体资料</p>
</li>
<li><p>答案直接关联到来源</p>
</li>
</ul>
<p><strong>从右到中：AI 建议 → 内容</strong></p>
<ul>
<li><p>AI 生成的内容直接进入 Crafts</p>
</li>
<li><p>可以立即编辑和调整</p>
</li>
<li><p>保持人机协作的灵活性</p>
</li>
</ul>

                    <h3 id="3-上下文的智能感知">
                        <a href="#3-上下文的智能感知">3. 上下文的智能感知</a>
                    </h3>
                <p>YouMind 的 AI 不是&quot;健忘&quot;的：</p>
<ul>
<li><p><strong>它知道你在哪个 Board</strong>：只会引用当前项目的资料</p>
</li>
<li><p><strong>它知道你在看什么</strong>：如果你正在阅读某个 Material，Chat 会优先参考它</p>
</li>
<li><p><strong>它记得对话历史</strong>：可以进行多轮深入讨论</p>
</li>
</ul>

                    <h2 id="实用技巧">
                        <a href="#实用技巧">实用技巧</a>
                    </h2>
                
                    <h3 id="高效使用-materials">
                        <a href="#高效使用-materials">高效使用 Materials</a>
                    </h3>
                <p><strong>✅ 做到这些：</strong></p>
<ul>
<li><p>保存资料后立即高亮重点（Picks）</p>
</li>
<li><p>添加你的想法（Notes）标注为什么保存这个</p>
</li>
<li><p>使用&quot;Listen&quot;功能生成音频，利用碎片时间理解</p>
</li>
</ul>
<p><strong>❌ 避免这些：</strong></p>
<ul>
<li><p>保存后不管，让资料堆积</p>
</li>
<li><p>不做标注，后面忘记为什么保存</p>
</li>
<li><p>收集过多，导致 Board 过于复杂（建议每个 Board 不超过 15 个 Materials）</p>
</li>
</ul>

                    <h3 id="高效使用-crafts">
                        <a href="#高效使用-crafts">高效使用 Crafts</a>
                    </h3>
                <p><strong>✅ 做到这些：</strong></p>
<ul>
<li><p>从模板或 AI 生成的大纲开始</p>
</li>
<li><p>利用右侧工具快速优化段落</p>
</li>
<li><p>保持&quot;人主导、AI 辅助&quot;的协作模式</p>
</li>
</ul>
<p><strong>❌ 避免这些：</strong></p>
<ul>
<li><p>盯着空白页不知道从哪开始（用模板或让 AI 生成初稿）</p>
</li>
<li><p>完全采用 AI 生成内容而不修改（失去个人观点）</p>
</li>
<li><p>忽略引用功能，导致内容无法追溯来源</p>
</li>
</ul>

                    <h3 id="高效使用-chattools">
                        <a href="#高效使用-chattools">高效使用 Chat/Tools</a>
                    </h3>
                <p><strong>✅ 做到这些：</strong></p>
<ul>
<li><p>提问时明确指出要参考哪些 Materials</p>
</li>
<li><p>尝试不同的 AI 模型（Claude 擅长分析，ChatGPT 擅长创意）</p>
</li>
<li><p>用 Chat 做头脑风暴：&quot;给我 5 个可能的写作角度&quot;</p>
</li>
</ul>
<p><strong>❌ 避免这些：</strong></p>
<ul>
<li><p>问过于宽泛的问题（如&quot;帮我研究远程工作&quot;）</p>
</li>
<li><p>只用默认模型，不尝试其他选择</p>
</li>
<li><p>把 Chat 当搜索引擎，而不是基于你的 Materials 深入讨论</p>
</li>
</ul>

                    <h2 id="界面的隐藏功能">
                        <a href="#界面的隐藏功能">界面的隐藏功能</a>
                    </h2>
                
                    <h3 id="1-顶部的quotaddquot按钮">
                        <a href="#1-顶部的quotaddquot按钮">1. 顶部的&quot;Add&quot;按钮</a>
                    </h3>
                <ul>
<li><p>快速添加新 Material</p>
</li>
<li><p>支持粘贴 URL、上传文件</p>
</li>
<li><p>不用离开当前视图</p>
</li>
</ul>

                    <h3 id="2-board-groups（分组功能）">
                        <a href="#2-board-groups（分组功能）">2. Board Groups（分组功能）</a>
                    </h3>
                <ul>
<li><p>在 Materials 区创建子文件夹</p>
</li>
<li><p>比如&quot;用户研究&quot;、&quot;竞品分析&quot;、&quot;设计稿&quot;</p>
</li>
<li><p>让大型项目更有条理</p>
</li>
</ul>

                    <h3 id="3-picks（高亮精选）">
                        <a href="#3-picks（高亮精选）">3. Picks（高亮精选）</a>
                    </h3>
                <ul>
<li><p>你在 Materials 中的所有高亮会单独收集</p>
</li>
<li><p>可以一键查看所有 Picks</p>
</li>
<li><p>写作时快速引用关键论点</p>
</li>
</ul>

                    <h3 id="4-历史版本">
                        <a href="#4-历史版本">4. 历史版本</a>
                    </h3>
                <ul>
<li><p>Crafts 会保存编辑历史</p>
</li>
<li><p>可以回溯到之前的版本</p>
</li>
<li><p>看到创作的演变过程</p>
</li>
</ul>

                    <h2 id="移动端的差异">
                        <a href="#移动端的差异">移动端的差异</a>
                    </h2>
                <p>如果你使用 YouMind 的移动 App：</p>
<ul>
<li><p>界面会调整为<strong>标签页模式</strong>（因为手机屏幕窄）</p>
</li>
<li><p>Materials、Crafts、Chat 变成独立的 Tab</p>
</li>
<li><p>但功能和关联性保持一致</p>
</li>
<li><p>可以通过分享菜单直接保存内容到 Board</p>
</li>
</ul>

                    <h2 id="最后的建议">
                        <a href="#最后的建议">最后的建议</a>
                    </h2>
                <p><strong>理解 YouMind 界面的关键：</strong></p>
<p>不要把它当成三个独立的工具，而是一个<strong>有机的创作系统</strong>：</p>
<ul>
<li><p><strong>Materials</strong> 是输入端</p>
</li>
<li><p><strong>Chat</strong> 是处理和思考端</p>
</li>
<li><p><strong>Crafts</strong> 是输出端</p>
</li>
</ul>
<p>它们之间的流动是自然的、双向的：</p>
<ul>
<li><p>你可以从 Materials 拖到 Crafts</p>
</li>
<li><p>也可以在 Crafts 中搜索 Materials</p>
</li>
<li><p>Chat 始终连接着两者，提供智能支持</p>
</li>
</ul>
<p><strong>现在试试这个流程：</strong></p>
<ol>
<li><p>打开你的 Board，看左侧有哪些 Materials</p>
</li>
<li><p>在右侧 Chat 问：&quot;这些资料的核心主题是什么？&quot;</p>
</li>
<li><p>在中间 Crafts 创建一个新文档，把 AI 的回答作为起点</p>
</li>
<li><p>从左侧拖拽 2-3 个重点到 Crafts，展开写作</p>
</li>
</ol>
<p>你会发现，三个区域的协作是如此流畅，思路不会被打断。这就是 YouMind &quot;集成创作环境&quot;的真正价值。</p>
