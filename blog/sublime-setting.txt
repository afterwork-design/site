                    <h2 id="Sublime 实时预览 Markdown 的方法">
                        <a href="#Sublime 实时预览 Markdown 的方法">Sublime 实时预览 Markdown 的方法</a>
                    </h2>
                <p>此方法是在浏览器中预览 Markdown，需要插件 <code>Markdown Preview</code> 和 <code>LiveReload</code> 。</p>
<ol>
<li>安装 <code>Markdown Preview</code> 插件。</li>
<li>设置预览快捷键。在 <code>Preferences</code> - <code>Key Bindings</code>中添加以下内容，即可通过快捷键 <code>Alt  m</code> 在浏览器中预览 Markdown：</li>
</ol>
<pre><code>[
    { &quot;keys&quot;: [&quot;alt+m&quot;], &quot;command&quot;: &quot;markdown_preview&quot;, &quot;args&quot;: {&quot;target&quot;: &quot;browser&quot;, &quot;parser&quot;:&quot;markdown&quot;}  }
]
</code></pre><ol start="3">
<li>安装 <code>LiveReload</code> 插件。不安装这个插件的话，文件修改后，需要手动刷新浏览器才能预览修改后的文件。</li>
<li>启用实时刷新预览。按快捷键 <code>Ctrl Shift P</code> ，输入 <code>LiveReload:Enable/disable plug-ins</code> ，选择 <code>Enable - Simple Reload</code> 。</li>
</ol>
