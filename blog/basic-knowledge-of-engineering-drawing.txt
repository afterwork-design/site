                    <h1 id="工程图基础知识">
                        <a href="#工程图基础知识">工程图基础知识</a>
                    </h1>
                
                    <h2 id="一般流程">
                        <a href="#一般流程">一般流程</a>
                    </h2>
                <p>工程图是将三维设计转换成二维投影的图。一套符合工程标准的图通常包含视图、标注、图纸三个方面。</p>
<p>操作流程：<strong>创建视图 → 调整视图 → 添加标注（线、尺寸、文本、符号、辅助草图、表格）</strong>，此外还需设置合适的图纸，以容纳视图和标注。</p>

                    <h2 id="视图">
                        <a href="#视图">视图</a>
                    </h2>
                <table>
<thead>
<tr>
<th>视图类型</th>
<th>概念</th>
<th>适用场景</th>
</tr>
</thead>
<tbody><tr>
<td>基础视图</td>
<td>从指定方向投影的三维模型视图</td>
<td>展示零件的主要外形和核心特征</td>
</tr>
<tr>
<td>投影视图</td>
<td>基于父视图的8个预设方向生成的投影视图</td>
<td>补充主视图未覆盖的俯视、侧视等标准视角</td>
</tr>
<tr>
<td>剖面视图</td>
<td>沿剖切线切割零件后生成的内部结构视图</td>
<td>展示孔、槽、腔体等隐藏结构</td>
</tr>
<tr>
<td>局部视图</td>
<td>放大父视图的指定区域</td>
<td>标注复杂或微小区域（如螺纹、凹槽）</td>
</tr>
<tr>
<td>辅助视图</td>
<td>垂直于父视图中选定边或面的特殊投影视图</td>
<td>表达倾斜面、曲面的真实形状和尺寸</td>
</tr>
</tbody></table>

                    <h3 id="基础视图">
                        <a href="#基础视图">基础视图</a>
                    </h3>
                <p>从一个指定方向上看到的三维模型的直接投影，可以是仰视图、俯视图、前视图、后视图、左视图、右视图、正等轴测图等。</p>

                    <h3 id="投影视图">
                        <a href="#投影视图">投影视图</a>
                    </h3>
                <p>相对于父视图的 8 个预设方向的投影视图。</p>

                    <h3 id="剖面视图">
                        <a href="#剖面视图">剖面视图</a>
                    </h3>
                <p>相对于父视图沿剖切线生成的视图，便于展示模型内部构造。</p>
<p>剖面视图涉及的元素：父视图、剖切线、剖面视图</p>
<ul>
<li><p>父视图：可以是基础视图、投影视图、辅助视图、局部视图</p>
</li>
<li><p>剖切线：可以是直线、阶梯线</p>
</li>
<li><p>剖面图：可以是全剖、半剖、阶梯剖、旋转剖</p>
</li>
</ul>

                    <h3 id="局部视图">
                        <a href="#局部视图">局部视图</a>
                    </h3>
                <p>放大显示的父视图某一区域的视图，用于精确标注复杂或微小区域的尺寸。</p>

                    <h3 id="辅助视图">
                        <a href="#辅助视图">辅助视图</a>
                    </h3>
                <p>垂直于父视图上选定边的投影视图，用于标注投影视图中无法表达的部分，如倾斜面、复杂曲面的形状。</p>

                    <h2 id="标注">
                        <a href="#标注">标注</a>
                    </h2>
                <p>几何图形：包含中心线、中心符号等元素</p>
<p>尺寸标注：包含长度、距离、角度、直径、半径、弧长等尺寸标注</p>
<p>注释：包含引线、文本等元素</p>
<p>符号：包含表面粗糙度、形位公差、基准标识符号、焊接、锥度和斜度等符号</p>
<p>草图：用草图元素绘制的图形，可以更灵活地进行标注。</p>
<p>表格：包含零部件明细表、折弯表、孔表等</p>

                    <h2 id="图纸">
                        <a href="#图纸">图纸</a>
                    </h2>
                <p>图纸信息包含图纸尺寸、标题、边框等。</p>
