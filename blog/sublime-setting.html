Sublime 实时预览 Markdown 的方法

此方法是在浏览器中预览 Markdown，需要插件 Markdown Preview 和 LiveReload 。

  1. 安装 Markdown Preview 插件。
  2. 设置预览快捷键。在 Preferences - Key Bindings中添加以下内容，即可通过快捷键 Alt m 在浏览器中预览 Markdown：
[
    { "keys": ["alt+m"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"}  }
]
  1. 安装 LiveReload 插件。不安装这个插件的话，文件修改后，需要手动刷新浏览器才能预览修改后的文件。
  2. 启用实时刷新预览。按快捷键 Ctrl Shift P ，输入 LiveReload:Enable/disable plug-ins ，选择 Enable - Simple Reload 。

更换设备时如何快速迁移配置

将原设备中 C:\Users\用户名\AppData\Roaming\Sublime Text 的 Installed Packages 和 Packages 文件夹复制到新设备的相同目录下即可。
