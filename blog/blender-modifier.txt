                    <h1 id="Blender 修改器">
                        <a href="#Blender 修改器">Blender 修改器</a>
                    </h1>
                <p>修改器是一种自动操作，以非破坏性的方式影响对象的几何形状。修改器的工作原理是改变对象的显示和渲染方式，而不是改变可以直接编辑的几何图形。</p>
<p><strong>添加修改器：</strong> <code>属性编辑器</code> &gt; <code>修改器属性</code> &gt; <code>添加修改器</code></p>
<p><strong>应用修改器：</strong> <code>Ctrl A</code>，应用多个修改器时从上到下应用才不会出错。</p>

                    <h2 id="修改器类型">
                        <a href="#修改器类型">修改器类型</a>
                    </h2>
                <p><img src="../assests/blog/image/modifier.jpg" alt="修改器"></p>
<p><strong>1.修改 Modify</strong></p>
<p>修改组修改器是类似于形变修改器，不过不会直接影响物体形状，但会影响物体的一些其他数据，比如顶点组。</p>
<p><strong>2.生成 Generate</strong></p>
<p>这些是建设性/破坏性的工具，能够影响整个网格的拓扑。它们可以改变对象的整体外观，或添加新的几何图形到对象。</p>
<p><strong>3.形变 Deform</strong></p>
<p>与生成不同，它们只改变对象的形状，而不改变其拓扑结构。</p>
<p><strong>4.模拟 Physics</strong></p>
<p>这些模拟代表的是物理模拟 。在大多数情况下，只要启用了粒子系统或物理模拟，它们就会自动添加到修改器堆队列中。它们唯一的作用是定义其在修改器堆队列中的位置，从这个位置获取它们所表示的模拟的基本数据。因此，它们通常没有属性，由属性编辑器中单独关于模拟的部分中的设置控制。</p>

                    <h2 id="表面细分修改器 Subdivision Surface Modifier">
                        <a href="#表面细分修改器 Subdivision Surface Modifier">表面细分修改器 Subdivision Surface Modifier</a>
                    </h2>
                <blockquote>
<p><a href="https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/subdivision_surface.html">https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/subdivision_surface.html</a></p>
</blockquote>
<p>表面细分修改器用于将网格的面分割成更小的面，使其看起来更平滑。</p>
<p>下图为细分等级分别为 0 到 3，无平滑着色和有平滑着色下的差异。</p>
<p><img src="../assests/blog/image/subdivision-surface.png" alt="表面细分修改器"></p>

                    <h3 id="基本操作">
                        <a href="#基本操作">基本操作</a>
                    </h3>
                <ol>
<li>选择模型，添加表面细分修改器。可以从修改器中选择表面细分修改器，也可以通过快捷键 <code>Ctrl</code> + <code>数字</code> 来添加对应细分等级的表面细分修改器</li>
<li>修改参数。</li>
</ol>

                    <h3 id="模式区别">
                        <a href="#模式区别">模式区别</a>
                    </h3>
                <ul>
<li>Catmull-Clark 型：细分并平滑曲面</li>
<li>简单型：仅细分网格，不平滑</li>
</ul>

                    <h3 id="相关快捷键">
                        <a href="#相关快捷键">相关快捷键</a>
                    </h3>
                <p>Ctrl 2：添加细分级数为 2 的表面细分修改器。</p>

                    <h3 id="常见问题">
                        <a href="#常见问题">常见问题</a>
                    </h3>
                <ol>
<li>Catmull-Clark 细分会对所有边进行磨圆，但部分边不想被磨圆</li>
</ol>
<p>有两种方法可以控制：</p>
<p>（1）加权边折痕
表面细分的加权边折痕允许您更改表面细分修改器细分几何体的方式，使边具有平滑或锐利的外观。</p>
<p>所选边的折痕权重可以在3D视图的侧栏的 变换 面板中更改。类似缩放工具的 Shift-E 也可以用来调整折痕的权重。数值越大，边缘越 锐利 ，对细分曲面的平滑效果越有抵抗力。</p>
<p>（2）加循环边</p>

                    <h2 id="实体化修改器 Solidify Modifier">
                        <a href="#实体化修改器 Solidify Modifier">实体化修改器 Solidify Modifier</a>
                    </h2>
                <blockquote>
<p><a href="https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/solidify.html">https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/solidify.html</a></p>
</blockquote>
<p>实体化修改器获取任意网格的表面，然后为之添加深度，使之变厚。</p>

                    <h2 id="倒角修改器 Bezel Modifier">
                        <a href="#倒角修改器 Bezel Modifier">倒角修改器 Bezel Modifier</a>
                    </h2>
                <blockquote>
<p><a href="https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/bevel.html">https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/bevel.html</a></p>
</blockquote>
<p>倒角修改器可以把应用该修改器的网格的棱切成斜角，允许控制如何以及哪里把倒角操作应用到网格上。</p>
<p>倒角修改器是编辑模式下 倒角操作 的非破坏性替代方式。</p>
<p><strong>宽度类型</strong></p>
<p>定义如何使用宽度以决定倒角总量。</p>
<p><img src="../assests/blog/image/bevel_width-methods.png" alt="倒角宽度类型"></p>
<ul>
<li>偏移 Offset：原始边到倒角面的垂直距离。</li>
<li>宽度 Width：坡口形成的两条新边的距离（如果有多个段，则坡口两侧的边）。</li>
<li>深度 Depth：该值是从原始边到倒角面的垂直距离。</li>
<li>百分比 Percent：新边线滑移距离相对邻边长度的百分比。</li>
<li>绝对 Absolute：沿着与斜面边缘相邻的边缘的确切距离。当连接到斜面边缘的非斜面边缘与直角以外的角度相遇时，可以看到与偏移的差异。</li>
</ul>

                    <h2 id="布尔修改器 Boolean Modifier">
                        <a href="#布尔修改器 Boolean Modifier">布尔修改器 Boolean Modifier</a>
                    </h2>
                <blockquote>
<p><a href="https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/booleans.html">https://docs.blender.org/manual/zh-hans/4.0/modeling/modifiers/generate/booleans.html</a></p>
</blockquote>
<p>布尔修改器可以对两个物体进行三种布尔运算（并集、差集、交集）。</p>
<p><img src="../assests/blog/image/three-bool.png" alt="三种布尔运算"></p>

                    <h2 id="蒙皮修改器 Skin Modifier">
                        <a href="#蒙皮修改器 Skin Modifier">蒙皮修改器 Skin Modifier</a>
                    </h2>
                
                    <h3 id="作用">
                        <a href="#作用">作用</a>
                    </h3>
                <p>蒙皮修改器使用顶点和边来创建蒙皮面，使用每个顶点的半径来更好地定义形状。</p>
<p><img src="../assests/blog/image/modeling_modifiers_generate_skin_panel.png" alt="蒙皮修改器参数"></p>

                    <h3 id="基本操作">
                        <a href="#基本操作">基本操作</a>
                    </h3>
                <ol>
<li>创建顶点。可以先创建一个任意网格物体，编辑模式下，<code>M</code> 合并顶点到中心</li>
<li>添加蒙皮修改器</li>
<li>挤出顶点。一个顶点可以挤出多个分支</li>
<li>标记根点。选择一个顶点，点击标记根点，根点外围会显示一圈红色虚线。每组连接的顶点都必须且只能有一个根点，默认会选择一个，也可手动更换根点</li>
<li>缩放蒙皮顶点半径。选中顶点，<code>Ctrl A</code> 进行缩放。注意快捷键不是 <code>S</code> ，而是 <code>Ctrl A</code>，<code>Ctrl A</code> 以后再按 <code>X/Y</code> 可以把缩放约束到某一个轴向上</li>
<li><code>Ctrl R</code> 可插入顶点</li>
</ol>

                    <h3 id="快捷键">
                        <a href="#快捷键">快捷键</a>
                    </h3>
                <ul>
<li><code>E</code>：挤出</li>
<li><code>Ctrl A</code>：缩放顶点半径</li>
<li><code>Ctrl R</code>：插入顶点。注意快捷键不是 <code>S</code> ，而是 <code>Ctrl A</code>，<code>Ctrl + A</code> 以后再按 <code>X/Y</code> 可以把缩放约束到某一个轴向上</li>
</ul>

                    <h2 id="置换修改器 Displace Modifier">
                        <a href="#置换修改器 Displace Modifier">置换修改器 Displace Modifier</a>
                    </h2>
                
                    <h3 id="作用">
                        <a href="#作用">作用</a>
                    </h3>
                <p>置换修改器根据纹理的强度置换网格中的顶点。可以使用程序纹理或图像纹理。</p>
<p>主要用来模拟凹凸效果。</p>
<p><img src="../assests/blog/image/modeling_modifiers_deform_displace_example.jpg" alt="置换修改器效果"></p>

                    <h3 id="基本操作">
                        <a href="#基本操作">基本操作</a>
                    </h3>
                <ol>
<li>添加平面，并表面细分。因为置换修改器是置换网格中的顶点，顶点太少置换效果就不明显，细分是为了增加顶点。可以在编辑模式下右键，选择细分，也可以添加表面细分修改器或多级精度修改器来实现细分</li>
<li>添加置换修改器</li>
<li>添加贴图。点击置换修改器面板中的 <code>新建</code>，转到纹理属性面板，选择一个纹理。Blender 内置了一些程序化纹理，也可以打开自定义图像纹理</li>
<li>调整置换修改器和贴图参数。</li>
</ol>

                    <h2 id="镜像修改器 Mirror Modifier">
                        <a href="#镜像修改器 Mirror Modifier">镜像修改器 Mirror Modifier</a>
                    </h2>
                
                    <h3 id="作用">
                        <a href="#作用">作用</a>
                    </h3>
                <p>镜像修改器沿着自身 X/Y/Z 轴，穿过物体原点镜像网格。还可以使用另一个物体作为镜像中心，使用该物体的局部轴向而非自身轴向。</p>

                    <h3 id="基本操作">
                        <a href="#基本操作">基本操作</a>
                    </h3>
                <ol>
<li>选择需要镜像的物体</li>
<li>添加镜像修改器</li>
<li>调整修改器参数</li>
</ol>

                    <h3 id="技术关键点">
                        <a href="#技术关键点">技术关键点</a>
                    </h3>
                <ol>
<li>镜像中心：默认为物体自身原点，也可设置为其他物体</li>
<li>镜像平面：过镜像中心垂直于所选轴的平面</li>
<li>切分：删除镜像平面一侧网格，可翻转</li>
<li>合并：对间距小于阈值的顶点进行合并</li>
</ol>
