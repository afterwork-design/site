                    <h1 id="Blender 基础操作">
                        <a href="#Blender 基础操作">Blender 基础操作</a>
                    </h1>
                <p>Blender 系列文章是我在学习 Blender 过程中总结的一些笔记，我使用的版本为 Blender 3.2，主要学习源为：</p>
<ol>
<li><a href="https://docs.blender.org/manual/zh-hans/4.0/index.html">官方文档 Blender 4.0 Manual</a></li>
<li><a href="https://www.bilibili.com/video/BV14u41147YH?p=1&vd_source=6546622e168eb1d92cd784a5dac6cfa6">中文教程</a></li>
<li><a href="https://www.youtube.com/playlist?list=PLgO2ChD7acqH5S3fCO1GbAJC55NeVaCCp">英文教程</a></li>
</ol>
<p>文章中快捷键均指的在 Windows 系统中，一般来说，在 macOS 系统中可使用 Cmd 替代 Ctrl 。</p>

                    <h2 id="控制视图">
                        <a href="#控制视图">控制视图</a>
                    </h2>
                <ul>
<li>旋转视角：中键拖拽</li>
<li>移动视角：Shift 中键拖拽</li>
<li>缩放视角：滚轮或 +、-</li>
</ul>
<p><strong>数字小键盘</strong></p>
<ul>
<li>7：顶视图，Ctrl 7：底视图</li>
<li>1：前视图，Ctrl 1：后视图</li>
<li>3：右视图，Ctrl 3：左视图</li>
<li>9：反转当前视图</li>
<li>5：切换透视、正交视图</li>
<li>0：摄像机视图</li>
<li>2/4/6/8：前后左右微调视角</li>
</ul>
<p>注：没有数字小键盘的话，可以使用字母键上方的数字键来模拟，但会导致使用不了这些键的原始功能（比如在编辑模式下快速切换点/线/面模式）。设置方法：
<code>偏好设置 (Preferences)</code> &gt; <code>输入 (Input)</code> &gt; <code>键盘 (Keyboard)</code> ，勾选 <code>模拟数字键盘 (Emulate Numpad)</code></p>
<p><img src="../assests/blog/image/emulate-numpad.jpg" alt="模拟数字键盘"></p>
<p><strong>调整摄像机位置角度</strong></p>
<p>先按 <code>N</code> 调出侧栏，在 <code>View</code> 面板中勾选 <code>Lock Camera to View</code>，再调整视图。调整好后记得取消勾选 <code>Lock Camera to View</code> 。</p>
<p><img src="../assests/blog/image/lock-camera.png" alt="锁定相机视图"></p>

                    <h2 id="控制物体">
                        <a href="#控制物体">控制物体</a>
                    </h2>
                <ul>
<li>移动物体：G</li>
<li>旋转物体：R</li>
<li>缩放物体：S</li>
<li>退出命令：右键 或 Esc</li>
<li>撤销变换：Alt G、Alt R、Alt S</li>
<li>限制方向：X、Y、Z</li>
<li>限制平面：Shift X、Shift Y、Shift Z</li>
<li>隐藏物体：H</li>
<li>显示物体：Alt H</li>
<li>隐藏未选中的物体：Shift H</li>
<li>添加物体：Shift A</li>
<li>复制物体：Shift D</li>
<li>删除物体：Delete 或 X</li>
</ul>
<p><strong>选择工具</strong></p>
<ul>
<li>调整：W</li>
<li>框选：B</li>
<li>刷选：C</li>
<li>套索：L 或 Ctrl 鼠标右键</li>
<li>循环选择模式：W</li>
<li>全选：A</li>
</ul>
<p>按住 Shift 加选、减选</p>

                    <h2 id="调整区域">
                        <a href="#调整区域">调整区域</a>
                    </h2>
                <p><strong>改变大小：</strong> 移动鼠标光标移到两个区域之间的边界，直到它变成一个双向箭头，然后按下左键并拖动。</p>
<p><img src="../assests/blog/image/areas_resize.png" alt="改变大小"></p>
<p><strong>拆分：</strong> 分割一个区域将创建一个新的区域。将鼠标光标放在一个区域的角上，光标会变成一个十字（+），然后按下左键就会激活分割或连接。从一个区域的角向内拖动将该区域拆分。你可以通过水平或垂直拖动来定义分割方向。</p>
<p><img src="../assests/blog/image/areas_split.png" alt="拆分"></p>
<p><strong>合并：</strong> 从一个区域的角落向外拖动，将两个区域连接起来。将被关闭的区域会显示一个较暗的浮动层，你可以通过将鼠标移到哪个区域来选择哪个区域将被关闭。释放左键来完成连接。如果你在释放鼠标前按下 Esc 或右键，操作将被取消。</p>
<p><img src="../assests/blog/image/areas_join.png" alt="合并"></p>
<p><strong>区域选项：</strong> 在区域边界按右键打开区域选项。可选择垂直分割、水平分割、合并区域、对调区域。</p>
<p><img src="../assests/blog/image/areas_options.png" alt="区域选项"></p>
<p><strong>切换区域最大化：</strong> Ctrl 空格</p>

                    <h2 id="模型编辑">
                        <a href="#模型编辑">模型编辑</a>
                    </h2>
                <ul>
<li>切换编辑/物体模式：Tab</li>
<li>编辑模式下切换点/线/面选择模式：1/2/3 （开启模拟数字键盘后，无法使用数字键切换选择模式，可直接点击对应的图标按钮进行切换）</li>
<li>切换透视/正常模式：Alt Z</li>
<li>应用：Ctrl A （尽量在编辑模式下使用缩放，避免在物体模式下使用缩放，如在物体模式下缩放，记得最后需执行应用。）</li>
<li>调出属性面板：F9 （生成物体后未进行其他操作时有效）</li>
<li>独立显示选中物体：/</li>
<li>合并模型：Ctrl J</li>
</ul>
<p><strong>点线面的选择</strong></p>
<ul>
<li>选择两个元素最短路径：Ctrl</li>
<li>选择相连元素：L</li>
<li>循环选择：按住 Ctrl 双击，就可选择一个循环边</li>
<li>垂直方向的循环选择：Ctrl Alt 双击</li>
</ul>
