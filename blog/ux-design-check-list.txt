                    <h1 id="体验设计走查表">

                    <h2 id="一信息结构与使用流程">
                        <a href="#一信息结构与使用流程">一、信息结构与使用流程</a>
                    </h2>
                
                    <h3 id="信息结构">
                        <a href="#信息结构">信息结构</a>
                    </h3>
                <ul>
<li>信息结构清晰易理解</li>
<li>导航间关系清晰易理解</li>
<li>页面信息层级清晰合理</li>
</ul>

                    <h3 id="使用流程">
                        <a href="#使用流程">使用流程</a>
                    </h3>
                <ul>
<li>新功能合理引导</li>
<li>相似任务中，用户体验路径一致</li>
<li>操作路径符合用户预期</li>
</ul>

                    <h2 id="二界面呈现">
                        <a href="#二界面呈现">二、界面呈现</a>
                    </h2>
                
                    <h3 id="样式">
                        <a href="#样式">样式</a>
                    </h3>
                <ul>
<li>Logo 清晰可识别，无变形</li>
<li>功能 icon 清晰、一致</li>
<li>字体字号合理、一致</li>
<li>色彩清晰、一致</li>
<li>元素间距合理、一致</li>
<li>插图清晰可识别</li>
</ul>

                    <h3 id="控件">
                        <a href="#控件">控件</a>
                    </h3>
                <ul>
<li>控件外观符合用户认知</li>
<li>控件状态完整</li>
<li>控件交互符合一致性</li>
</ul>

                    <h3 id="数据输入">
                        <a href="#数据输入">数据输入</a>
                    </h3>
                <ul>
<li>相关字段和标签合理分组</li>
<li>阅读路径顺畅</li>
<li>阅读无歧义</li>
<li>注明可选/必填字段</li>
<li>数据格式、单位明确</li>
</ul>

                    <h2 id="三交互体验">
                        <a href="#三交互体验">三、交互体验</a>
                    </h2>
                
                    <h3 id="交互过程与反馈">
                        <a href="#交互过程与反馈">交互过程与反馈</a>
                    </h3>
                <ul>

<li>操作失败有反馈，并提供解释与方案</li>
<li>危险操作二次确认，可撤销</li>
</ul>
