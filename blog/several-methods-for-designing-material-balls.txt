                    <h1 id="设计材质球的几种思路">
                        <a href="#设计材质球的几种思路">设计材质球的几种思路</a>
                    </h1>
                <p>最近的 UI 设计工作中，需要用到多种材质渲染图，要这些图片既可以作为图标，又可作为材质预览图，所以至少要 2 套不同尺寸且背景透明的材质渲染图。</p>
<p>我探索了几种方式来实现，整个过程也是用 AI 辅助 UI 设计的一次成功尝试。</p>

                    <h2 id="手绘效果图">
                        <a href="#手绘效果图">手绘效果图</a>
                    </h2>
                <p>我在 <a href="https://tangweijuan.com/blog/figma-3d-icon">Figma 设计轻 3D 效果图标教程</a> 中曾经讲过用 Figma 做 3D 效果，总体思路就是将想要的效果分解成多个图层，通过给每一层设置不同的颜色、阴影、模糊等参数，模拟简单的 3D 效果，但这次需要用到的效果比较复杂，且后续会扩展更多复杂的材质，所以暂不考虑手绘效果图。</p>
<p><img src="../assests/blog/image/inflate.jpg" alt="inflate"></p>

                    <h2 id="手动建模渲染">
                        <a href="#手动建模渲染">手动建模渲染</a>
                    </h2>
                <p>手动建模和渲染，我在 Spline 和 Blender 两个软件里都试过，都可以实现，且可控性高，但速度较慢，需要查材质参数、调环境、调灯光、调相机一系列操作。</p>
<p><img src="../assests/blog/image/spline-material.png" alt="spline-material"></p>

                    <h2 id="AI 直接生图">
                        <a href="#AI 直接生图">AI 直接生图</a>
                    </h2>
                <p>自己手动调过一遍参数后，想着有没有更快的方式呢，于是想到了 AI。我让 ChatGPT 直接帮我生成材质渲染图，结果金属效果都不错，非金属尤其是玻璃材质，需要几次调整才可以，所以这种用自然语言描述的方式还是让我不放心，效果不可控，且可扩展性低。</p>
<p><img src="../assests/blog/image/chatgpt-material.png" alt="chatgpt-material"></p>

                    <h2 id="AI 辅助建模渲染">
                        <a href="#AI 辅助建模渲染">AI 辅助建模渲染</a>
                    </h2>
                <p>那如果不让 AI 直接生成图片，而是辅助我建模渲染呢？于是我问 ChatGPT：</p>
<blockquote>
<p>我想在 Blender 中创建多种金属材质球，并分别导出背景透明的 PNG 图片，该如何建模，如何设置灯光、相机等?</p>
</blockquote>
<p>没想到它不仅给了我操作步骤，还提供了其他方案：</p>
<blockquote>
<ul>
<li>提供一套现成的 Blender 文件模板</li>
<li>或生成不同金属参数设置参考表（色值 + metallic/roughness 建议）</li>
<li>或写一个完整的 .blend 自动化脚本</li>
</ul>
</blockquote>
<p>我选择了使用自动化脚本。运行一次脚本，就创建出了我指定的十几个材质球。灯光和相机有些问题，在我反馈后，也很快就调整正确了。</p>
<p><img src="../assests/blog/image/adjust-1.png" alt="adjust-1"></p>
<p><img src="../assests/blog/image/adjust-2.png" alt="adjust-2"></p>
<p><img src="../assests/blog/image/adjust-3.png" alt="adjust-3"></p>
<p>Blender 脚本使用方式：</p>
<ul>
<li>打开 Blender 文本编辑器</li>
<li>粘贴脚本</li>
<li>运行脚本</li>
<li>切换到相机视图（Numpad 0 或视图中 View &gt; Cameras &gt; Active Camera）</li>
<li>在“材质属性面板”中自由切换球体材质</li>
</ul>
<p>最终效果如下：</p>
<p><img src="../assests/blog/image/auto-blender-material.png" alt="auto-blender-material"></p>
