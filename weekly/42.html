Vol.42：如何区分春天那些花?

你好，欢迎阅读我的周刊第 42 期。在这里，我将分享近期我读到的、看到的有趣或有用的东西，也分享我的一些灵感和想法。

本周主题：如何区分梅花、樱花、桃花、李花、梨花、杏花？

最近正值春天，是百花盛开的季节，周末去公园逛了逛，发现有些花长得很像，如果不是看标志牌，我还真分不清，所以回来后就查了一些资料，找到了每一种花的特点。

**梅花：**冬末春初开花，没有花柄，开花时不见叶，花的香味浓，花萼呈红褐色，花瓣为倒卵形，白或粉红色。

**樱花：**成簇开放，花瓣尖端有缺口。早樱的花蕾粉色卵圆形，花粉白色，花丝浅紫红色，小枝淡紫褐色，叶片椭圆卵形或倒卵形。而晚樱的花瓣粉色倒卵形，叶片为椭圆状卵形，小枝灰白色或淡褐色。樱花花期在3~4月。樱花种类非常多。

**桃花：**开花比长叶要早一些，盛花期时，叶子就会长出来了，花瓣粉红或深红，在春日里绽放着浓郁的光彩。

**李花：**成簇开放，花柄很长，刚开花的时候也不见叶，花瓣洁白，花朵小而细碎。有一个说法是李白的名字就是在和父亲作诗时得来的。

在李白七岁那年的春天，父亲看着春日院落中葱翠树木，似锦繁花，开口吟诗道：“春国送暖百花开，迎春绽金它先来。”母亲接着道：“火烧叶林红霞落”。李白知道父母吟了诗句的前三句，故意留下最后一句，希望自己接续下去。他走到正在盛开的李树花前，稍稍想了一下说：“李花怒放一树白”。“白”——不正说出了李花的圣洁高雅吗？父亲灵机一动，决定把妙句的头尾“李” “白”二字选作孩子的名字，便为七岁的儿子取名为“李白”。

**梨花：**成簇开放，花瓣洁白如雪，花蕊略带红色，花和叶子都比较大，边开花边长叶。梨树一般比较高大，树皮银白，枝干嶙峋苍劲。

**杏花：**单朵开放，先开花后长叶子，有短花梗或者无花梗，花瓣为白色或粉红色，5 瓣。杏在全国各地都有栽种，花期为3～4月。杏花的独有特点是萼片是紫绿色的，而且花开了之后它就会反折。

微软推出 Microsoft 365 Copilot

2023 年 3 月 16 日，微软宣布推出 Microsoft 365 Copilot，将 Copilot 嵌入到 Word、Excel、PowerPoint、Outlook、Teams 等中，以释放创造力、提高生产力。

  • Word：能够在人们工作时与他们一起撰写、编辑、总结和创作。
  • PowerPoint：能够在创作过程中，通过自然语言命令将想法转化为设计好的演示文稿。
  • Excel：能够帮助用户释放洞察、识别趋势，或在短时间内创建专业型式的数据可视化。
  • Outlook：能够帮助用户整合并管理收件箱，从而节约出更多时间用于实际沟通。
  • Teams：能够直接从对话上下文中提供实时摘要和待办事项，提高会议效率。
  • Power Platform：通过在 Power Apps 和 Power Virtual Agents 中引入两项新功能，能够让各种技能水平的开发人员利用低代码工具加速和简化开发。
  • Business Chat：汇集了来自文档、演示文稿、电子邮件、日历、笔记和联系人的数据，能够帮助用户总结聊天内容、撰写电子邮件、查找关键日期，甚至根据其他项目文件制定计划。

一图了解 Web3.0

现在到处都在提到 Web 3.0，那么它到底是什么呢，这个思维导图提供了一些科普。

什么是 Web3.0：

Web 1.0 是只读信息（1990-2004年）；

Web 2.0 是读+写信息（2004年-至今）；

Web 3.0 是读+写+拥有信息（2014年-至今）

Web 3.0 的核心技术：

区块链；人工智能；语义网；边缘计算；3D 图形和虚拟现实技术

Web 3.0 关键名词：

DFT (Non-fungible Token)：独一无二、完全不同，难以用来平等交换的代币；

DAO (Decentralized Autonomous Organization)：去中心化自治组织；

Dapp (Decentralized Applications)：去中心化应用；

DeFi (Decentralized Finance)：去中心化金融系统。
