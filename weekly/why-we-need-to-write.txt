                    <h1 id="Vol.6：为什么普通人也需要写作？">
                        <a href="#Vol.6：为什么普通人也需要写作？">Vol.6：为什么普通人也需要写作？</a>
                    </h1>
                <p><img src="https://tangweijuan.github.io/weekly/image/%E4%B8%BA%E4%BB%80%E4%B9%88%E6%99%AE%E9%80%9A%E4%BA%BA%E4%B9%9F%E9%9C%80%E8%A6%81%E5%86%99%E4%BD%9C.jpg" alt="为什么普通人也需要写作"></p>
<p>当说到写作，你可能会问，“我又不是作家，为什么要写作？”</p>
<p>你可能还会想，“我也不再是学生了，更没必要写作了，难道上学时还没被作文逼疯吗？”</p>
<p>在我决定写博客前，我也是这样想的，一个设计师，干嘛要写作呢。在经历了一段时间的探索后，才发现了写作的意义。</p>

                    <h3 id="1. 写作，就是一次和自己的对话过程">
                        <a href="#1. 写作，就是一次和自己的对话过程">1. 写作，就是一次和自己的对话过程</a>
                    </h3>
                <p>对我来说，开始想要写作地契机是因为有一段时间觉得自己状态不好，于是我决定找一些自我管理类的书籍来看，看着看着就产生了共鸣，有时遇到一些事，就在记事本里写下来，真实地记录当时的心情和想法，看到写下的文字时，才发现原来自己对一些事情的看法是这样的。</p>

                    <h3 id="2. “如果你说不清楚，就表示你自己也不明白”">
                        <a href="#2. “如果你说不清楚，就表示你自己也不明白”">2. “如果你说不清楚，就表示你自己也不明白”</a>
                    </h3>
                <p>John Searle的这个观点在一定程度上我是认同的，当我能用自己的话将一个知识讲清楚时，那才是真正的学会了。</p>

                    <h3 id="3. 记录生活">
                        <a href="#3. 记录生活">3. 记录生活</a>
                    </h3>
                <p>那些去过的地方、吃过的美食、看过的电影……几年之后，你还记得吗？当你如实地写下自己的经历，在多年后再看到这些文字，你能知道，哦，原来生活还是很丰富多彩的。</p>

                    <h3 id="4. 分享与交流知识">
                        <a href="#4. 分享与交流知识">4. 分享与交流知识</a>
                    </h3>
                <p>当我们在学习知识的过程中，将自己的收获写下来，再分享给别人，让别人也学到更多，这样利己又利人的事，又何乐而不为呢。分享出去的内容，如果得到了反馈，那就更是一件值得的事了。别人的反馈，有肯定也会有否定，在肯定的反馈里，可以勉励我们继续；在否定的反馈里，则可以从多角度思考，从而拓展自己的思维。</p>

                    <h3 id="5. 进行深度学习">
                        <a href="#5. 进行深度学习">5. 进行深度学习</a>
                    </h3>
                <p>当我们需要写作时，必定会先学习相关知识，然后将书本中的知识结合自己的实践，去除糟粕，取其精华，当最终能再教授给他人时，我们得到的将远超预期。</p>
