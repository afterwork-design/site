                    <h1 id="Vol.3：生活美学杂志、杂志网站、杂志app推荐">
                        <a href="#Vol.3：生活美学杂志、杂志网站、杂志app推荐">Vol.3：生活美学杂志、杂志网站、杂志app推荐</a>
                    </h1>
                <p><img src="https://tangweijuan.github.io/weekly/image/cover-%E6%9D%82%E5%BF%97.jpg" alt="cover-杂志"></p>

                    <h3 id="1.%20%3Ca%20href=%22https://www.wallpaper.com/%22%3EWallpaper*%3C/a%3E">
                        <a href="#1.%20%3Ca%20href=%22https://www.wallpaper.com/%22%3EWallpaper*%3C/a%3E">1. <a href="https://www.wallpaper.com/">Wallpaper*</a></a>
                    </h3>
                <p>一本英国杂志，由Tyler Brule于1996年在伦敦创立，涵盖建筑、室内家居、旅行和美食等方面内容，创刊后很快就成为了众多设计师、建筑师的美学灵感杂志。</p>
<p><img src="https://tangweijuan.github.io/weekly/image/wallpaper.jpg" alt="wallpaper"></p>
<p>2017年11月 Wallpaper* 走进了中国，中国刊由栩栩华生创刊，名为《 Wallpaper* 卷宗》，杂志采用双月发行的方式，首刊封面是在松美术馆拍摄的，刘雯担任模特，整个封面极具中国古典美学。</p>
<p><img src="https://tangweijuan.github.io/weekly/image/wallpaper%E5%8D%B7%E5%AE%97.jpg" alt="wallpaper卷宗"></p>

                    <h3 id="2.%20%3Ca%20href=%22http://voicer.me%22%3EVoicer%3C/a%3E">
                        <a href="#2.%20%3Ca%20href=%22http://voicer.me%22%3EVoicer%3C/a%3E">2. <a href="http://voicer.me">Voicer</a></a>
                    </h3>
                <p><img src="https://tangweijuan.github.io/weekly/image/voicer.jpg" alt="voicer"></p>
<p>一个独立品牌线上杂志，用他们自己的介绍来说就是：</p>
<blockquote>
<p>以多元开放的视角，分享生活和设计的美学，探索未来生活与创造的可能性。</p>
</blockquote>
<p>他们也有自己的公众号，可以在手机上方便地获得最新内容，喜欢听播客的还可以收听他们的播客「吃茶」。</p>

                    <h3 id="3. Holiday">
                        <a href="#3. Holiday">3. Holiday</a>
                    </h3>
                <p><img src="https://tangweijuan.github.io/weekly/image/holiday.jpg" alt="holiday"></p>
<p>从Voicer知道的这个杂志：“从前有一本美国杂志，嚣张到令人发指：派摄影师公费拍外景，不指定地点，一走就是一年半载；而内文阵容有多梦幻呢？随便翻开一篇专题，作者就是海明威本人。”</p>
<p>原文在这里 <a href="https://mp.weixin.qq.com/s/ZwLJBIupKp2l7-hzbfNzaQ">如何不上班？这本杂志研究了70年……</a></p>

                    <h3 id="4.%20%3Ca%20href=%22https://www.nowness.cn/%22%3ENOWNESS%3C/a%3E">
                        <a href="#4.%20%3Ca%20href=%22https://www.nowness.cn/%22%3ENOWNESS%3C/a%3E">4. <a href="https://www.nowness.cn/">NOWNESS</a></a>
                    </h3>
                <p><img src="https://tangweijuan.github.io/weekly/image/nowness.jpg" alt="nowness"></p>
<p>一个国际创意短片平台，主要以视频为载体，展现当代艺术与设计、时尚与美容、音乐、美食与旅行、文化等内容，每一个短片质量都非常高。</p>

                    <h3 id="5.%20%3Ca%20href=%22https://www.abitare.it/en/%22%3EABITARE%3C/a%3E">
                        <a href="#5.%20%3Ca%20href=%22https://www.abitare.it/en/%22%3EABITARE%3C/a%3E">5. <a href="https://www.abitare.it/en/">ABITARE</a></a>
                    </h3>
                <p><img src="https://tangweijuan.github.io/weekly/image/abitare.jpg" alt="abitare">
源于意大利的建筑杂志，在建筑、设计、人文等方面都有很深度的解析。</p>
